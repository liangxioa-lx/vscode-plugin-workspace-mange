# workspace-mange

## 功能说明 (Function description)

> 这是一个 vscode 插件，用于管理工作区，支持添加、删除、移动、编辑工作区，支持多窗口同时打开。

> This is a vscode plug-in for managing workspaces, support to add, delete, move, edit workspaces, support multiple Windows open at the same time.

## 使用说明 （How to use）

> 安装完插件后，点击侧边栏图标进入插件，点击右上角的“添加”按钮，输入工作区名称，回车键确认即可添加目录至插件中；
> 点击列表中的目录即可新窗口打开该目录，如果目录已被打开，则会聚焦到该目录的 vscode 窗口；
> 点击列表中的目录，右键菜单中可以进行删除、编辑、移动等操作；


> After the plug-in is installed, click the sidebar icon to enter the plug-in, click the "Add" button in the upper right corner, enter the workspace name, and press Enter to confirm to add the directory to the plug-in;
> Click the directory in the list to open the directory in a new window. If the directory is already open, it will focus on the vscode window of the directory;
> Click the directory in the list, right-click the menu to delete, edit, move and other operations;

## 注意事项 (Attention)

> 当多个窗口打开时，某窗口对目录列表进行了修改操作，其他窗口不会实时同步，这时仅需点击其他窗口插件中的刷新按钮即可同步最新的目录配置；

> When multiple Windows are open, the directory list is modified in one window, and other Windows will not be synchronized in real time. In this case, you only need to click the refresh button in other window plug-ins to synchronize the latest directory configuration.
